# xting
## It is a personal music player

### Dependency:
* python >= 3.7
* mutagen
* PyQt6 (python and Qt6 bindings)

### Run:
* Navigate to source dir in console and type: `python xting.py` in console to run the application
* or directly click `xting.py` to run
